# JSR 303 

This project is a example and study case of the JSR 303, here you will see the integration with Spring Boot + JAX-RS

Examples:

- Simple validations
- Custom validations
- Composed validations
- Composed validations with single violation
- Group validations

#### Tools/Frameworks

- Java8
- Spring Boot
- JAX-RS (Jersey impl)
- JSR-303 (Hibernate impl)
- Maven
- Rest-Assured
- Junit
- Mockito
